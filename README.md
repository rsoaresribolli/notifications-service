# Install Prisma CLI
```
npm i prisma -D
```

# Install Prisma Client
```
npm i @prisma/client
```

# Set Prisma Datasource
```
npx prisma init --datasource-provider SQLite
```

# Run migrations
```
npx prisma migrate dev
```

# Open Prisma Interface
```
npx prisma studio
```

# Install validators
```
npm install class-validator class-transformer
```
